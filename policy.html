Política de Privacidade - DevBot

O DevBot é um plugin de integração com o GitHub desenvolvido para uso pessoal e profissional.

Informações Coletadas

O DevBot apenas manipula dados dos repositórios do usuário no GitHub conforme as ações solicitadas. Nenhum dado é armazenado em servidores externos.

Uso de Informações

As informações são usadas exclusivamente para a execução das ações no GitHub solicitadas pelos usuários.

Segurança

O DevBot utiliza autenticação por token para acesso ao GitHub, seguindo as melhores práticas de segurança da API.

Contato

Em caso de dúvidas ou solicitações relacionadas à privacidade, entre em contato pelo e-mail: suporte@bnzhost.com.br

Alterações

Esta política pode ser atualizada sem aviso prévio para melhor refletir o compromisso com a segurança e privacidade dos usuários.
